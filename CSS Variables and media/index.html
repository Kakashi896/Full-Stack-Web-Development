~ Media Queries

Media queries are used when we want to customize our website's presentation according to the user's screen size. With the help of media queries, user can display different markups based upon the device's general type(mobile, desktop, tablet). It is a logical operation. Whenever a media query becomes true, then the related CSS is appliedto the target element. Syntax: @media media-type and (media-feature) @media signifies the start of a media query, the type property defines the display screen onwhich the rules are to be applied like phone, tablet, pc, etc. Media-feature tells about the dimensions of the screen.
